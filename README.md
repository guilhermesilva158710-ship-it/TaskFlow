# TaskFlow

## Descrição
Sistema simples de gerenciamento de tarefas desenvolvido utilizando Python.

## Funcionalidades
- Criar tarefas
- Listar tarefas
- Atualizar tarefas
- Excluir tarefas
- Filtrar tarefas

## Tecnologias
- Python
- Pytest
- GitHub Actions

## Metodologia
Kanban

## Estrutura
/src
/tests
/docs

## Como executar
python src/main.py

## Mudança de Escopo
Foi adicionada a funcionalidade de filtro de tarefas por status para melhorar a organização do sistema.
